# Changelog

All notable changes to Spotify Deck will be documented in this file.

This project follows semantic versioning.

## [Unreleased]

### Added

- Spotify app window button using a static local launch command.
- Per-player volume slider backed by MPRIS.
- Mouse-wheel volume adjustment over the widget.
- Release packaging script for `.plasmoid` downloads.
- GitHub Actions workflow for build artifacts and tagged releases.

## [0.0.1] - 2026-04-30

### Added

- Initial KDE Plasma 6 applet.
- Transparent Rockies-purple UI with black accents.
- Current track and artist display.
- Animated wave bars during playback.
- Previous, play/pause, and next controls.
- Local-only MPRIS/DBus security documentation.
